### 문제
영어 알파벳으로 이루어진 문자열 str이 주어집니다. 각 알파벳을 대문자는 소문자로 소문자는 대문자로 변환해서 출력하는 코드를 작성해 보세요.

- 입력
    
    aBcDeFg

- 출력
    
    AbCdEfG

### 제한사항
1 ≤ str의 길이 ≤ 20

str은 알파벳으로 이루어진 문자열입니다.

### ⭐배열 안의 문자를 아스키코드를 이용해 대소문자를 구분하고 변환하도록(charCodeAt) 구현.  <br/> join 메서드를 이용하여 배열 형태의 문자를 문자열 형태로 출력하게 함.
